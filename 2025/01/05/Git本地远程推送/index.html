Windows 下载：winget install Git.Git

Linux 下载

  • Debian/Ubuntu：sudo apt install git

  • CentOS/Fedora：sudo yum install git 或 sudo dnf install git

  • Arch/Manjaro：sudo pacman -S git

  • MacOS 下载：brew install git
