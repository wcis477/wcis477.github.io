安装

内容参考Arch WiKiArch提供了自动化安装脚本可以输入archinstall进行配置

下载Arch镜像

验证ISO文件密钥SHA256

Arch 官网下载界面密钥在这查看

制作启动盘

进入Rufus 官网下载rufus启动盘制作软件分区类型GPT，目标系统类型UEFI

如果是固态U盘就打开显示高级设备选项

勾选上显示USB外置硬盘

验证引导模式

连接WIFI

rfkill unblock wifi解除网络接口软件屏蔽

Name是适配器名称，SSID是无线网络名称

连接上了可使用 ping archlinux.org测试网络是否通畅

更新系统时间

创建硬盘分区

fdisk创建分区

我的分区方案是一个EFI分区、一个SWAP分区、一个/(根)分区

输入 g 创建一个GPT分区表输入 n 创建分区

接下来依次是分区编号、起始扇区、结束扇区分区编号如果是默认直接回车即可、起始扇区也是结束扇区根据想要分区多大输入，1G就输入 +1G

512MB就输入 +512M输入 p显示分区表输入 w保存分区表写入磁盘输入 q放弃更改后的分区表

格式化分区

挂载分区

遵循先挂载到根分区，再挂载EFI分区，最后挂载其他分区

配置软件源

进入 /etc/pacma.d/mirrorlist文件找到想要的软件源把前面的#去掉保存即可

安装必需软件包

pacstrap -K /mnt base linux linux-firmware安装linux内核文件显示分区表如果计算机是intel的CPU追加intel-ucode，AMD CPU追加amd-ucode文件管理系统的用户工具：

ext4 e2fsprogsexfat exfatprogsBtrfs btrfs-progsXFS xfsprogs

访问RAID分区的工具 mdadm访问LVM分区的工具 lvm2内建音频的 sof-firmware,用于Marvell无线的 linux-firmware-marvellBroadcom wireless的固件包 broadcom-wl联网所需的程序 modemmanager文件编辑器如 nano,vim访问man和info页面中文档的工具:man-db,man-pages,texinfossh工具 openssh、网络管理器 networkmanagersudo sudo、zsh zsh、git git、base-level base-level、nodejs nodejs、npm

生成fstab文件

chroot到新安装的系统

设置时区

区域化本地设置

打开/etc/locale.gen文件取消en_US.UTF-8 UFT-8和zh_CN.UTF-8 UTF-8的注释，保存退出执行locale-gen命令打开/etc/locale.conf文件添加LANG=en_US.UTF-8和LC_TIME=zh_CN.UTF-8两行

网络配置

设置root密码

安装Grub引导程序

安装软件包efibootmgr、grub

进入cfg文件 找到 set timeout=10把10改为0保存退出

撤出启动盘进入系统后使用 nmtui工具连接网络

创建普通用户

使用root用户在 /etc/sudoers文件里找到 root ALL=(ALL:ALL) ALL复制在下一行粘贴，把root换成我们要添加的普通用户名，保存退出（这样就能普通用户就能使用sudo命令执行root权限了）建议用户名不要用中文、大写，否则Hyprland登陆会登不进去

配置

配置ArchLinuxCN

使用清华的软件源在/etc/pacman.conf 末尾添加以下语句

添加本地信任GPG keysudo pacman-key --lsign-key "farseerc@archlinux.org"安装archlinuxcn密钥sudo pacman -Sy archlinuxcn-keyring配置git的全局设置

npm换源

使用npm下载工具需要root权限

Github SSH下载

登录github，点击头像，点击设置，找到 SSH and GPG keys 选项，点击 SSH keys 右边的 New SSH key 按钮，标题给个恰当的密钥名字，把 cat ~/.ssh/id_ed25519.pub的输出全部复制进 Key 里面，单击 Add SSH key，接着输入你的 github 登录密码ssh -T git@github.com弹出以下内容，代表成功Hi your_name! You’ve successfully authenticated, but GitHub does not provide shell access.

paru安装

sudo vim /etc/pacman.conf把 Color的注释去掉sudo nano /etc/paru.conf 把BottomUp注释去掉paru -Syu 更新

Hyprland安装

安装过程需要全程联网并且建议挑网络好的时候安装如果安装过程中经常有软件下载失败可以先把最后的标记失败的软件安装上，再配置好npm和github ssh下载方式，再进行下载

安装一些娱乐软件

安装中文字体

oh-my-zsh

如果安装完毕，终端没有ohmyzsh美化，可以自己尝试安装

LazyVim

保证装有neovim和git

电池管理

风扇控制

ufw防火墙

安装VSCode

在VSCode下载链接下载.tar.gz包

把安装包tar -zxvf 安装包.tar.gz解压，放在/opt/目录下

使用ln命令创建软件链接ln -s /opt/code-stable-x64-1731511985 /opt/vscode

创建vscode.desktop文件(快捷方式)链接

把文件放在 /usr/share/applications/目录下，结束
