Hexo基本操作

Hexo基本操作

一月 09, 2025
  • Hexo

下载Hexo

npm install -g hexo-cli

创建博客

hexo init blog

创建文章

hexo new text

创建页面

hexo new page test

预览和部署网站

预览网站

hexo clean && hexo g && hexo s

部署网站

npm install hexo-deployer-git

在根目录下_config.yml配置:

hexo clean && hexo g && hexo d
